Ducumentation

Projects

Use the following metadata for new posts.

---
title: Some Title
description: Some Description 
image: some-image.png
layout: post
categories: some-category, another-category
---

Place images in the images folder and use its name in the above frontmatter where image: some-image.img is defined.

Reading

Use the following metadata for new posts.

---
layout: post
title: Book Title
date: 2015-09-06
book-cover: some-image.gif
rating: 4
---

Place images in the images folder and use its name in the above frontmatter where book-cover: some-image.gif is defined.

Thinking

Use the following metadata for new posts.

---
layout: post
title: Some-title
categories: some-category, another-category
---

Misc

Use the following metadata for new posts.

---
layout: post
title:  Some Title
description: Some description
img: image-8.jpg
author: author-name
categories: category
tags: tag
---

Place images in the images folder and use its name in the above frontmatter where img: image-8.jpg is defined.

Don't use two categories or two tags which may result in undesired outputs.
